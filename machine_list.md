# 服务器机器列表
方便各位玩家准备材料时寻找需要的机器

### 小黑塔

- **原理：** 利用远离地面部分小黑刷新位置限制，末影螨吸引仇恨，将小黑砸致残血后轻易击杀
- **主要产物：** 经验
- **副产物：** 末影珍珠
- **传送石碑：** `enderman generator`
- **使用指南：** 等待大量小黑掉落后挥砍，将需要修复的物品放置在副手（装备会自动修复）
- **修复：** 本机器几乎没有损坏可能
- **运行状态：** 长期运行（已稳定运行1年+）
- **建造者：** leipishu、StormMessager、Ryan_Fengrh

### 刷沙机

- **原理：** 利用末地传送门框架在特定时机下对重力方块的状态判定漏洞。活塞将沙子推入末地传送门框架的角落，在下落被阻断的瞬间，游戏既判定它为“已放置的方块”，又保留其“下落实体”的属性。此时玩家激活传送门，系统强制更新方块状态，导致这个被卡住的沙子被“复制”——一份被传送门带走并消失，另一份则因实体残留而以物品形式掉落，从而实现了沙子的凭空复制。
- **主要产物：** 沙子/混凝土/沙砾
- **副产物：** 黑曜石（假人手动挖掘）
- **传送石碑：** `stronghold`（主世界部分）`the end`（末地部分）
- **使用指南：** 将需要刷五的重力方块放置在传送门边粘液块上方，打开放置边的对应拉杆，并在主世界挂机，离开时一定要关机！！！
- **修复：** 无法找到原教程，损坏请联系建造者
- **运行状态：** 关机（可随时开机）
- **建造者：** Ryan_Fengrh

### 潜影贝农场

- **原理：** 利用潜影贝被导弹击中时产生新个体的复制机制，通过时序电路控制“母体”与“复制体”循环攻击，将新生贝自动收集并运送至处死区，从而稳定产出潜影壳。整个过程围绕精准的复制触发和物流运输设计，依赖矿车与红石时序实现自动化循环。
- **产物：** 潜影壳
- **传送石碑：** `shulker`
- **使用指南：** 区块加载即工作，在靠近末地城一侧有挂机平台提升效率（爬上脚手架后关闭两层活扳门，可使用假人），在机器本体靠末地城一侧最下方的箱子里可以取用打包好的潜影壳
- **修复：** 原作者：balllight(ballight_xy) [前往→](https://www.bilibili.com/video/BV1nucReJErm/?spm_id_from=333.337.search-card.all.click)
- **运行状态：** 长期运行（稳定性待验证，定期补充打包机所需潜影盒）
- **建造者：** leipishu、Ryan_Fengrh

### 刷铁机

- **原理：** 利用村民构成村庄后受僵尸恐吓会召唤铁傀儡并处死铁傀儡刷铁
- **主要产物：** 铁锭
- **副产物：** 虞美人
- **传送石碑：** `industry`
- **使用指南：** 区块加载即工作,从楼梯下是收集部分，建议定期合成铁块缓解存储压力避免浪费
- **修复：** 搬运作者：图酱 [前往→](https://bilibili.com/video/BV1Rg4y1775m/?spm_id_from=333.337.search-card.all.click)
- **运行状态：** 长期运行（已稳定运行1年+）
- **建造者：** leipishu

### 守卫者农场

- **原理：** 利用海底神殿自然刷新守卫者及玩家周围128格范围刷怪/消失特性，使守卫者刷新在灵魂沙平台上并由气泡和水流送至下界基岩上层处死
- **主要产物：** 海晶碎片、海晶沙砾，鳕鱼
- - **副产物：** 墨囊
- **传送石碑：** `守卫者农场`
- **使用指南：** 挂机即工作，最佳挂机点：传送石碑所在平台的脚手架上方
